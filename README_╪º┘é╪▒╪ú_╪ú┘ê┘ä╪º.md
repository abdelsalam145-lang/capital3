# ⚠️ اقرأ هذا أولاً — حل مشكلة "يحمّل ملف بدل ما يفتح"

## لماذا حدثت المشكلة؟

المستودع السابق كان فيه ملفات مختلطة: ملفات الخلفية (Laravel) + ملفات الواجهة (Vite) معاً، وفي الجذر مباشرة. كما أن ملف `main.jsx` رُفع باسم خاطئ (`download`). لذلك Vercel لم يجد بنية مشروع Vite صحيحة، ففشل البناء وحمّل ملفاً بدل عرض الصفحة.

## الحل: استخدم هذا المجلد وحده

هذا المجلد (`capital-dashboard`) يحتوي **مشروع الواجهة فقط، نظيفاً ومرتّباً**. ارفعه وحده.

البنية الصحيحة التي يجب أن تظهر في **جذر** المستودع (وليس داخل مجلد فرعي):

```
index.html          ← في الجذر مباشرة
package.json         ← في الجذر مباشرة
vite.config.js
vercel.json
src/
  main.jsx           ← الاسم الصحيح (ليس download)
  Dashboard.jsx
public/
  logo.png
```

## الطريقة الموصى بها (الأنظف) — مستودع جديد عبر Git

> ملاحظة: يبدو أنك أنشأت مستودعاً جديداً `capital2`. ممتاز — سنرفع عليه بشكل صحيح.

على جهازك (أو جهاز مبرمجك)، بعد فك ضغط هذا المجلد:

```bash
cd capital-dashboard          # ادخل مجلد المشروع النظيف
git init
git add .
git commit -m "لوحة مؤشرات نظيفة - بنية Vite صحيحة"
git branch -M main
git remote add origin https://github.com/abdelsalam145-lang/capital2.git
git push -u origin main
```

> إن قال إن المستودع غير فارغ، أضِف `--force` لأول دفعة فقط:
> `git push -u origin main --force`

## إن كنت سترفع يدوياً عبر موقع GitHub (سحب وإفلات)

مهم جداً: لا ترفع المجلد `capital-dashboard` نفسه. بل **ادخله، وحدّد كل ما بداخله** (`index.html`, `package.json`, `src`, `public`, ...) وارفع **المحتويات** — بحيث يكون `index.html` في جذر المستودع مباشرة، لا داخل مجلد.

## بعد الرفع على Vercel

1. في Vercel احذف المشروع القديم الفاشل (أو أنشئ مشروعاً جديداً).
2. `Add New → Project` → اختر مستودع `capital2`.
3. Vercel سيتعرّف تلقائياً على Vite. اضغط **Deploy**.
4. خلال دقيقة ستحصل على رابط يفتح اللوحة فعلياً (لا يحمّل ملفاً).

## كيف تتأكد أن الرفع صحيح قبل النشر؟

افتح المستودع على GitHub. يجب أن ترى `index.html` و `package.json` و `src/` ظاهرة مباشرة في الصفحة الرئيسية للمستودع. إن رأيت مجلداً واحداً فقط اسمه `capital-dashboard`، فالبنية خاطئة — ادخله وأعد الرفع بمحتوياته في الجذر.
